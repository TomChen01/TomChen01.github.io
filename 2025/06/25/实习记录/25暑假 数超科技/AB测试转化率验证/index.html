Project Record: Kids Camera PDP CVR Optimization

点击下载数据和代码

1. Project Brief & Hypothesis

1.1. Project Overview

  • Test Factors: Main Image Optimization for CVR Uplift
  • Method: A/B Test
  • ASIN(s) to be Tested: B0DCFQMXW5
  • Current BSR Ranking: #58 in Kids’ Cameras
  • Background: The current conversion rate for this ASIN presents an opportunity for improvement. We hypothesize that enhancing the main image set will lead to a significant uplift in conversion by better communicating the product’s value and features.
  • Objective: To increase the overall sales and profitability of the target ASIN.

1.2. Hypothesis

  • If… we change the main image from showing a variety of functions (including photography, video, music, games, etc.) to showing a variety of accessories (such as 32GB memory card, printing paper, multi-color brushes, etc.).
  • Then… we predict that the product’s conversion rate will increase by 5 percentage points.
  • Because… the design ideas and performance data of the main pictures in the top 15 BSRs show that this type of main picture has a better effect. This may be because the rich accessories can show the sincerity of the merchant, making users feel that they get more by buying one, thereby improving CVR.

2. Basic Settings

2.1. Metrics

  • Primary Metric:
    • Unit Session Percentage (Conversion Rate): To measure the impact of the image change.
    • Target: A relative uplift of 5% or more.
  • Secondary Metrics:
    • Sessions: To monitor for changes in traffic.
    • Page Views: To monitor user engagement.
    • Revenue: To measure the overall business impact.

2.2. Sample Size Estimation

The minimum sample size required to detect a meaningful difference (Absolute +5 percentage points) in CVR:

  • Baseline CVR: 9.14%
  • Expected CVR: ~ 14%
  • Delta: 0.05
  • Significance level ($\alpha$): 0.05
  • Power: 0.8

Using the sample size fomula of Two-Proportion Z-test:

$$
n = \frac{2(p_1(1-p_1) + p_2(1-p_2)) \cdot (Z_{1-\alpha/2} + Z_{1-\beta})^2}{(p_2 - p_1)^2}
$$

Where:

  • $p_1$：Baseline CVR
  • $p_2$：Experimental Group CVR
  • $Z_{1-\alpha/2} = 1.96$：Critical value for two-tailed test (confidence)
  • $Z_{1-\beta} = 0.84$：Value corresponding to 80% power

We can get the following results:

$$
\begin{align*}
\text{Required per group} &= 1,000 \text{ sessions} \\
\text{Total} &= 2,000 \text{ sessions}
\end{align*}
$$

Conclusion: Experiment is feasible with current traffic.

3. Experiment Design

3.1. Objective

To determine whether updating the main image leads to an absolute increase of 5 percentage points in conversion rate (from 9.14% to 14.14%) over a 14-day period.

3.2. Design Summary

Item Description
Test Duration 14 days
Traffic Split 50% Control (A), 50% Variant (B)
Group Assignment By user session
Controlled Elements A+ content, pricing, reviews
Primary Metric Conversion Rate (CVR)
Minimum Sample Size ~1,000 sessions per group
Actual Sample Size ~24,000+ sessions per group (sufficient)

3.3. Data Collection Strategy

Aggregate daily session and order data for both groups over 14 days:

Date Variant Sessions Orders CVR (%)
06-25 A 1,720 160 9.30%
06-25 B 1,725 230 13.33%
… … … … …
Total A 24,000 2,180 9.08%
Total B 24,230 3,400 14.02%

This structure supports reliable statistical analysis with minimized day-to-day variance.

A/B testing can be conducted directly by using tools integrated into the Amazon platform. The steps are as follows:

  • Open the brand function of the Amazon seller backend and select “Manage your experiments”;
  • All current experiments and their results will be > displayed on the page. Click “Create a new experiment”;
  • Select the title, main image or A+ from the drop-down list;
  • Select from the eligible listing ASIN.

Another method is to conduct AB testing between different variants. Although it requires strict control of variables, it has the following advantages:

  • Sellers who are not in the Amazon Brand Registry Program can also use this function.
  • The diversion ratio can be adjusted.

4. Data Collection & Cleaning

4.1. Data Collection

Data was collected over 14 days from Amazon traffic split evenly across the control (original image) and variant (new image) groups. For each day, we recorded the number of sessions and orders per group.

4.2. Data Cleaning

Method: python pandas, statsmodels…

  • Removed rows with null or missing session/order data
  • Verified that group labels and date formats were consistent
  • Checked for outliers and anomalies in CVR trends

5. Results

5.1. Statistics

The A/B test was conducted over a 14-day period, comparing the original main image (Group A) with an updated version (Group B). The goal was to assess whether the updated main image could improve the conversion rate.

Group Total Clicks Total Orders Overall CVR (%)
A 721.41754 69 9.56
B 826.05402 107 12.95

Group B achieved a higher conversion rate (12.95%) compared to Group A (9.56%), suggesting a positive trend following the main image update.

5.2. Statistical Test: Z-Test

To determine whether the observed improvement in conversion rate was statistically significant, a two-proportion Z-test was conducted.

The results were as follows:

$$
\text{Z-statistic} = -2.0945 \\
\text{P-value} = 0.0362
$$

Given that the p-value is less than the significance level of 0.05, we conclude that the difference is statistically significant. This indicates that the change in the main image had a real, positive impact on the conversion rate.

5.3. Visualization

The following charts show the daily trends for Conversion Rate (CVR) and Click-Through Rate (CTR):

Figure 1: Daily CVR for Group A and B — Group B shows a consistently higher CVR across most of the 14-day test period.

Figure 2: Daily CTR — patterns are generally stable with minor fluctuations between the two groups.

These visual insights reinforce the statistical findings, showing a clearer and more consistent advantage in Group B.

6. Final Recommendation

6.1. Interpretation

This A/B test set out with a clearly defined goal: to increase the conversion rate (CVR) of a key Amazon product listing by 5 percentage points through updating the main product image. Over the 14-day testing period, the control group (Group A) maintained a baseline CVR of 9.56%, while the treatment group (Group B) with the updated image achieved a CVR of 12.95%, representing a 3.81 percentage point increase.

Although the final improvement fell short of the target uplift, the difference between the two groups was statistically significant (p = 0.0362), confirming that the visual change had a real, measurable impact on user behavior.

The 3.81% absolute lift in conversion rate corresponds to a ~39.85% relative improvement over the baseline, which is substantial in an e-commerce context. Even modest increases in CVR can result in considerable revenue gains, especially when scaled across high-traffic listings.

6.2. Conclusions

We recommend adopting the updated main image for the product listing and continuing with a systematic experimentation roadmap. Future A/B tests can build on this momentum by targeting other high-impact visual and informational components.
